🔗 STEP 2 — 영상 불러오기
1
유튜브 영상 URL 붙여넣기 → 📥 불러오기 클릭
2
미답글 댓글을 자동으로 수집합니다 (최대 100개)
3
대댓글 달 댓글을 클릭하여 선택
✨ STEP 3 — AI 대댓글 생성
1
댓글 선택 후 ✨ 대댓글 생성하기 클릭
2
AI가 댓글을 분석하고 대댓글 3가지를 자동 생성합니다
3
마음에 드는 버전 📋 복사 → 유튜브에 붙여넣기
⚡ STEP 4 — 일괄 생성
1
영상 로드 후 ⚡ 미답글 일괄 생성 섹션 확인
2
🚀 일괄 생성 시작 → 미답글 전체 한 번에 처리
3
결과 테이블에서 복사 또는 ⬇️ CSV 저장
